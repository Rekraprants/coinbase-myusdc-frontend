## Prerequisites

- Git
- NodeJs
- pnpm

## Getting Started

- Install dependencies

```sh
pnpm i
```

- Fill in the environment variables in the `.env` file, refer to the `.env.example` file for the required variables.
- Make sure [configs](./src/configs/) is updated

## Run Dev

```sh
pnpm run dev
```

## Build Prod

```sh
pnpm run build
```
